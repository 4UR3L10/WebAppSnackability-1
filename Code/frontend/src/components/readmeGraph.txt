Component: snack-graph.component.js
Dependencies: graph.css -> Code/frontend/styles/graph.css
Title: SnackGraph
Latest Update: 07/13/2021

This component uses react-chartjs to create a graph based on the snacks consumed by the user for the past five days of snacks consumed. 

Data can be adjusted on setChartData and options for the appearance of the graph can be updated on const options function.
